# Chat Application

This is a simple real-time chat application developed as part of my internship task.  
The project demonstrates basic frontend and backend integration for real-time communication.

## Project Overview

The application is divided into two main parts:
- Frontend: User interface for chatting
- Backend: Server-side logic and real-time communication

Users can send and receive messages instantly.

## Technologies Used

### Frontend
- HTML
- CSS
- JavaScript
- React

### Backend
- Node.js
- Express.js
- Socket.io

## How to Run the Project

### Step 1: Run the Backend
Open a terminal and run:
cd backend
npm install
npm start
The backend server will start running on the specified port.

### Step 2: Run the Frontend
Open a **new terminal** and run:
cd frontend
npm install
npm start
The frontend application will open in the browser.
